Seminário Position Display

O position é uma propriedade CSS que define como um elemento deve ser posicionado
na página, ou seja, no corpo do documento o body. Essa propriedade possui quatro
propriedades que são top, bottom, right e left, respectivamente, topo, abaixo, direita e esquerda.
Que determinam a localização final do elemento, que irá se deslocar conforme especificado
na estilização. Além dessas propriedades, o position possui cinco tipos atribuídos a ele,
que determinam como o elemento será posicionado de acordo com a página. Que são: static, fixed, relative, absolute e sticky.

O CSS display é uma propriedade utilizada para organizar os elementos na página HTML. Ela pode conter diferentes valores, entre eles o flex e o grid, que possibilitam a criação de layouts responsivos.

Valores display:

Inline

Apresenta o elemento como conteúdo corrido,
inserido organicamente junto ao texto ou outros elementos,
ocupando apenas sua própria dimensão na tela, sem que propriedades
como width ou height provocam qualquer efeito.

Valores display:

Block

Apresenta o elemento como um bloco, inserido separadamente do conteúdo
e ocupado (até que se altere) 100% do espaço disponível na tela. None: não exibe o elemento na tela.

Valores display:

Inline-Block

Apresenta o elemento como conteúdo corrido, inserido organicamente
junto ao texto ou outros elementos, mas permite que propriedades como
width ou height o modifiquem como se fosse um tipo block

Valores display:

Grid

Apresenta o elemento como um contêiner de grid,
tipologia disponível apenas no HTML5 e em alguns navegadores.

Valores display:

Flex

Define o elemento como um contêiner flexível,
facilitando o alinhamento de itens filhos.

Valores position:

Static

O valor padrão de position é static, que mantém o elemento
no fluxo normal da página e não permite usar top, right, bottom ou
left para posicionamento. Outros valores, como relative e fixed, permitem alterar a posição do elemento.

Valores position:

Relative

O valor relative significa "relativo". Um elemento
com "position: relative" também é posicionado de acordo com o fluxo
normal da página, mas agora podemos usar as propriedades top, right, bottom e left para
ajustar sua posição relativa à sua posição original. Exemplo: .elemento

Exemplo -> position: relative;
top: 10px; /* Move o elemento 10 pixels para baixo */
left: 20px; /* Move o elemento 20 pixels para a direita */

Valores position:

Absolute

O absolute position é a ferramenta que remove o elemento do fluxo normal,
não afeta e também não é afetada pela posição dos outros elementos.
Diferente dos outros elementos no body, você pode posicionar qualquer elemento absoluto
dentro de um elemento se o mesmo estiver definido como algo diferente de static (O elemento que é posicionado conforme
o fluxo normal do html). Para definir o posicionameto é possível utilizar top left, right e bottom.

Valores position:

Fixed

O fixed position é, de maneira simples, elementos posicionados em relação a área de visualização
ou a janela do navegador, ou seja, ao rolar a página para baixo, os elementos fixos permanecem
fixos em suas posições. Essa configuração de posição é frequentemente usada para navegações fixas.
Vale salientar que, um elemento fixo se comporta de forma relativa ao fluxo do documento até atingir
uma posição de rolagem definida. Nesse ponto, ele passa a se comportar como um elemento fixo dentro de seu elemento pai direto.
Assim que o elemento fixo atinge a parte inferior do elemento pai, ele para de rolar.

Valores position:

Sticky

Para falar do sticky position é preciso também falar do fixed e do relative, pois o sticky é a junção dos dois elementos,
ele funciona como um fixed, fixa alguma imagem, texto ou elemento, porém, até um ponto em que o desenvolvedor usando o comando top: 0 estipula,
o que torna o elemento fixo até o final do contêiner, travando quando atinge o ponto definido.
