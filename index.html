Seminário Position Display

O position é uma propriedade CSS que define como um elemento deve ser posicionado
na página, ou seja, no corpo do documento o body. Essa propriedade possui quatro
propriedades que são top, bottom, right e left, respectivamente, topo, abaixo, direita e esquerda.
Que determinam a localização final do elemento, que irá se deslocar conforme especificado
na estilização. Além dessas propriedades, o position possui cinco tipos atribuídos a ele,
que determinam como o elemento será posicionado de acordo com a página. Que são: static, fixed, relative, absolute e sticky.

O CSS display é uma propriedade utilizada para organizar os elementos na página HTML. Ela pode conter diferentes valores, entre eles o flex e o grid, que possibilitam a criação de layouts responsivos.

Valores display: Inline

Apresenta o elemento como conteúdo corrido,
inserido organicamente junto ao texto ou outros elementos,
ocupando apenas sua própria dimensão na tela, sem que propriedades
como width ou height provocam qualquer efeito.

Valores display: Block

Apresenta o elemento como um bloco, inserido separadamente do conteúdo
e ocupado (até que se altere) 100% do espaço disponível na tela. None: não exibe o elemento na tela.

Valores display: Inline-Block

Apresenta o elemento como conteúdo corrido, inserido organicamente
junto ao texto ou outros elementos, mas permite que propriedades como
width ou height o modifiquem como se fosse um tipo block

Valores display: Grid

Apresenta o elemento como um contêiner de grid,
tipologia disponível apenas no HTML5 e em alguns navegadores.

Valores display: Grid

Define o elemento como um contêiner flexível,
facilitando o alinhamento de itens filhos.
